Wrangle and Analyze WeRateDogs Twitter data

Author: Satvinder Kaur Pujari

Date: 06/24/2020

This report explains the insights produced by wrangling, analyzing and visualizing the tweet archive of Twitter user @dog_rates , also known as WeRateDogs. WeRateDogs is a Twitter account that rates people's dogs with a humorous comment about the dog. These ratings almost always have a denominator of 10 and the numerators almost always greater than 10 (Ex: 11/10, 12/10, 13/10, etc.) because they think dogs are good. WeRateDogs has over 8 million followers and has received international media coverage.

WeRateDogs twitter archive data was collected from following three different sources:

 

1.      Enhanced twitter archive in csv file

2.      Image predictions file from Udacity webpage

3.      Twitter API

 

After cleaning and combining the collected data a master dataset twitter_archive_master.csv is formed for WeRateDogs.

 

Finally, after analyzing and visualizing this master dataset, following insights were drawn:

 

1.     Over the time period, WeRateDogs increased the rating values.

3.     In general, tweets with high rated dogs get more favorite counts. Dog ratings in the tweet posted and the number of favorite counts given to that tweet are positively correlated.

 

 

Limitations:

 

         The insights drawn from analyzing 'twitter_archive_master.csv' dataset are strictly based on the observations made by cleaning and visualizing different characteristics of this dataset. Any inferential statistical methods are not used to make the inferences or conclusions.

 

         Many parameter values in this dataset are missing which might have affected the analysis results.

 

         Few rare cases of rating styles were ignored while making above conclusions.

 

         There could be still be some inaccurate values in dataset based on the extraction techniques used to form twitter archive dataset.

 

         Image prediction file data is used to predict the dogs breed. The accuracy of this prediction depends on the accuracy of prediction algorithm used to gather the file data.

 

         To form this master dataset, three different tables were combined with some missing records in two tables. Because, gathering those records was not possible.
